# Changelog

## 0.2.1 - 2026-07-19

- 修复取消编辑后重新打开仍保留未保存掉落行的问题。
- 修复关闭期间替换数据集后编辑器仍使用旧草稿的问题。
- 补齐推荐宝鉴、副本明细和命中装备三类表格的行联动回归测试。
- 区分掉落行“已核对”和“未核对”按钮状态，保持实色操作按钮的可读性。

## 0.2.0 - 2026-07-19

- 将可见 UI 全面迁移到 HeroUI v3 和 Tailwind CSS v4。
- 重构三栏桌面工作台、筛选侧栏、条件摘要、结果 Tabs、三类结果表格和推荐详情。
- 增加亮色、暗色、跟随系统三档主题，并持久化用户选择。
- 重构固定 720px 掉落表编辑器，副本、宝鉴、部位和属性统一使用下拉组件。
- 增加权重正数校验、固定页脚和独立滚动区域。
- 修复下拉列表所有选项错误显示勾选标记的问题。
- 修复点击结果表格后右侧详情不更新的问题，并统一三类表格的行操作。
- 优化方角表头、满宽结果 Tabs、紧凑详情卡片和三栏整体阴影。
- 提高编辑器保存、核对和删除按钮对比度，并压缩弹框头部与底部高度。
- 删除不再使用的 shadcn/Radix 包装组件和直接依赖。

## 0.1.4 - 2026-07-19

- 更名为“诛仙高手秘境掉落软件”。
- 明确声明借鉴“诛仙世界-秘境掉落筛选”，并兼容其 `drop_tables.zx` 数据结构。
- 默认加载原版 5 个副本、40 组副本/宝鉴和 252 条装备掉落数据。
- 补齐原版 10 个属性筛选项，双属性按钮按完整组合匹配。
- 优化结果 Tab、推荐宝鉴表格、副本明细表和命中装备表。
- 数据编辑器改为固定高度，并使用下拉组件编辑副本、宝鉴、部位和属性。

## 0.1.3 - 2026-07-19

- 增加真实 `zx1` `.zx` 导出按钮，编辑后的数据可回写旧程序。

## 0.1.2 - 2026-07-19

- 统一 Tauri、Cargo、npm 和安装包版本号。

## 0.1.1 - 2026-07-19

- 完成旧版 `zx1` 加密协议兼容。
- 支持旧版 `version/dungeons/baojian/tables` 目录数据结构。
- 修复属性筛选后显示零命中宝鉴的问题。
- 生成 Windows NSIS 安装包。
