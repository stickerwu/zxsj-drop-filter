# 诛仙世界秘境掉落筛选设计

## 目标

以 Tauri 2、React、TypeScript 和 shadcn/ui 重做现有 Windows 掉落筛选工具，覆盖筛选、推荐宝鉴、概率计算、命中明细、掉落表编辑、JSON 导入和复制摘要。

## 已确认设计

- 主窗口使用三栏工作台：左侧筛选、中央结果表、右侧明细。
- 默认使用浅色“青玉纸本”主题，青玉色表达选中和成功状态。
- 属性匹配支持“命中任一”和“同一词条全满足”；空筛选表示不限。
- 中央结果包含推荐宝鉴、副本 × 宝鉴明细、命中装备列表三个视图。
- 数据编辑器以可调整大小的弹窗承载，支持新增、核对、修改、删除和应用当前表。
- 公开仓库使用 MIT；仅提供演示数据，真实 `.zx` 文件本机加载并忽略。

## 计算规则

属性组合先展开为基础集合。任一模式使用集合交集，全满足模式要求所选集合是词条集合的子集。每个副本和宝鉴分别计算总权重、命中权重、命中概率和期望次数；推荐按最佳概率、平均概率、命中条数和名称稳定排序。概率只在展示层格式化。

## 架构

Tauri 负责桌面窗口和本地能力，React 负责 UI，纯 TypeScript domain 层负责数据模型、匹配、推荐、摘要和序列化。JSON 是公开格式，`.zx` 是兼容边界。
